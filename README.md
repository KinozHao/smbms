## 超市订单管理系统
本项目基于javaWeb+jsp+Mysql+Tomcat+前端。

教程:[狂神说:JavaWeb](https://www.bilibili.com/video/BV12J411M7Sj?p=30)
### 开发环境
1. IDEA v2020.1.3
2. Tomcat v10.0.22
3. Maven v3.6.3
4. Mysql v5.7.20
5. git v2.35.1
